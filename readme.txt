# Projet Tang-air

This project is a web application developed as part of a flight management and passenger rating project. It allows users to search for flights, make reservations, rate passengers, and view flight informations.

## Features

- Flight search by departure airports and dates.
- Seat reservation on available flights.
- Passenger rating after the flight.
- Detailed information about flights, including airports, pilots, and aircraft.
- User and account management.

## Technologies Used

- Python: The main programming language used for the backend development.
- Flask: Python framework for web application development.
- SQLite: Database management system used to store flight, user, and rating information.
- HTML/CSS: Markup languages used for creating web pages and styling them.
- JavaScript: Programming language used for client-side dynamic content manipulation.

## Configuration and Installation

1. Clone this repository to your local machine: `git clone https://github.com/Seikonolff/Projet-Tang-air.git`
2. Make sure you have Python installed on your machine.
3. Install the required dependencies by running the following command: `pip install -r requirements.txt`
4. Run the application using the command: `python app.py`
5. Access the application in your browser by entering the URL: `http://localhost:5000`
6. Make sure to have a stable internet connection to enjoy the site
